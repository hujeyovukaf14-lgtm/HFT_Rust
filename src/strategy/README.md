# Strategy Module

Здесь находится "мозг" торгового робота.

## Как это работает?

Стратегия работает внутри бесконечного цикла Hot Thread.

### Signal (`signal.rs`)

*   **Логика:** Сравнивает цены с разных бирж (Binance vs Bybit).
*   **Триггер:** Решение принимается за несколько наносекунд на основе последних данных в `OrderBook`. Никаких сложных вычислений, блокирующих поток.

### Book Manager (`book_manager.rs`)

*   **Инкрементальные обновления:** Применяет дельта-апдейты (diffs) к локальному стакану.
*   **Reset:** Если обнаружен разрыв в `u` (update ID), модуль инициирует процедуру сброса (или Safe Mode), так как целостность данных нарушена.

**Важно:** Этот код никогда не должен паниковать. `.unwrap()` запрещен.
