# HFT Rust Project

Этот проект реализует высокочастотную (HFT) арбитражную стратегию на Rust, оптимизированную для AWS c8i.large (1 физическое ядро + HT).

## Запуск (Windows vs Linux)

Проект поддерживает условную компиляцию для разработки на Windows и запуска на серверах Linux.

### Windows (Development & Testing)

*   **Режим:** `DEV_MODE = true` (в `src/strategy/risk.rs`).
*   **Оптимизации:** `TCP_QUICKACK` отключен. Проверки латентности (Latency Checks) смягчены, паника при их нарушении отключена.
*   **Pinning:** Попытка привязать потоки к ядрам выполняется, но ошибки игнорируются (выводится warning), так как Windows Scheduler может блокировать это для обычных пользователей.

### Linux (AWS Production)

*   **Режим:** `DEV_MODE = false`. **ВНИМАНИЕ: Перед деплоем измените константу вручную!**
*   **Оптимизации:** Включаются `TCP_NODELAY` и `TCP_QUICKACK` (если поддерживается).
*   **Safety:** Любая задержка обработки тика > 50мкс вызовет **Panic** и перезагрузку процесса (Fail Fast).
